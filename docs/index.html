comToolCss

一些常用到的 css 样式整合成类选择器使用。

安装

使用 jsdelivr 的 cdn： https://cdn.jsdelivr.net/gh/tk914/comToolCss@v0.2.0/commonTool.min.css

备用：https://fastly.jsdelivr.net/gh/tk914/comToolCss@0.2.0/commonTool.min.css

Install with npm:

npm i comtoolcss --save

网页说明文档：（使用 markdown 转换工具 i5ting_toc）

https://tk914.github.io/comToolCss/

使用 scss 语法说明：

$T: ".com-";

使用

清除浮动

com-clearfix

具体代码：
*zoom: 1;
  &::after {
    clear: both;
  }
  &::before,
  &::after {
    content: "";
    display: table;
  }

左浮动

com-float-left

具体代码：
float: left;

右浮动

com-float-right

具体代码：
float: right;

强制显示竖向滚动条

com-overflow-scroll

具体代码：
overflow: scroll;
overflow-x: hidden;

水平居中

com-margin-center

具体代码：
margin: 0 auto;

绝对定位

com-absolute

具体代码：
position: absolute;

绝对定位水平垂直居中

com-absolute-center

具体代码：
position: absolute;
top: 50%;
left: 50%;
transform: translate(-50%, -50%);

绝对定位垂直居中

com-absolute-center-y

具体代码：
position: absolute;
top: 50%;
transform: translateY(-50%);

绝对定位水平居中

com-absolute-center-x

具体代码：
position: absolute;
left: 50%;
transform: translateX(-50%);

禁止用户选中

com-user-none

具体代码：
-webkit-user-select: none;
-moz-user-select: none;
-ms-user-select: none;
user-select: none;

溢出隐藏

com-overflow-hidden

具体代码：
overflow: hidden;

默认鼠标样式

com-mouse-default

具体代码：
cursor: default;

手形鼠标

com-mouse-point

具体代码：
cursor: pointer;

容器宽高等同屏幕大小

com-overspread

具体代码：
  width: 100vw;
  height: 100vh;

转为行级块

com-display-ib

具体代码：
  display: inline-block;

flex 布局

com-flex

具体代码：
  display: -webkit-box;
  display: -ms-flexbox;
  display: flex;

flex 布局主轴子元素居中

com-flex-main-center

具体代码：
  @extend #{$T}flex;
  -webkit-box-pack: center;
  -ms-flex-pack: center;
  justify-content: center;

flex 布局子元素水平垂直居中

com-flex-center

具体代码：
  @extend #{$T}flex-main-center;
  -webkit-box-align: center;
  -ms-flex-align: center;
  align-items: center;
  -ms-flex-line-pack: center;
  align-content: center;
